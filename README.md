# 🤖 NEURA – AI Interviewer

NEURA is an AI-powered virtual interviewer that conducts Technical and HR interviews using Generative AI, Resume Analysis, Computer Vision, and Voice Interaction.

The platform simulates a real interview environment where an AI interviewer communicates with the candidate, analyzes the uploaded resume, asks intelligent questions, and monitors for unfair practices using webcam-based proctoring.

---

# 🚀 Features

## ✅ AI Resume Analysis
- Upload PDF resume
- Extracts candidate skills and technologies
- Generates interview questions dynamically

## ✅ Technical & HR Interviews
- Technical interview mode
- HR interview mode
- Human-like conversational flow

## ✅ AI Voice Interaction
- AI asks questions with voice output
- Real-time conversation simulation

## ✅ Candidate Monitoring
- Webcam-based verification
- Detects:
  - No face
  - Multiple faces
  - Suspicious behavior

## ✅ Intelligent Interview Flow
- Context-aware questioning
- Follow-up questions
- Resume-based discussion

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/NEURA.git
cd NEURA
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Add API Key

Create a `.env` file:

```env
OPENAI_API_KEY=YOUR_API_KEY
```

---

## 4️⃣ Run Project

```bash
python app.py
```
